# Part 1 — База мест + карточка места  (V1)

**Цель:** показать одно место так, чтобы помочь принять решение и добавить в борд.
**Зависит от:** contracts: `Place` (+ позже `Verdict` из Part 3 для блока вердикта).
**Контракт вход → выход:** вход `Place` → выход UI карточки + событие `addToBoard(placeId)`.
**Собрать:**
- Hero-фото + галерея 3–5 кадров; кнопка «в борд» прямо на фото.
- Блоки: почему особенное, лучшее время, сложность+время, как добраться, с ребёнком, безопасность (+ дата актуальности), похожие места.
- CTA-бар: В борд · Я был · В маршрут · Поделиться (кнопки могут быть заглушками до своих частей).
**Definition of done:** карточка рендерит все блоки из мок-`Place`; есть тест, что пустые поля не ломают вёрстку.
**Не загружать:** реализацию квиза, борда, маршрутов — только их типы при необходимости.
