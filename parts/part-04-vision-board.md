# Part 4 — Vision Board (ядро)  (V1)

**Цель:** высокочастотный объект мечты: красивая доска мест с умными метками.
**Зависит от:** contracts: `Board`, `BoardItem`, `Place` (+ `Verdict` для меток).
**Контракт вход → выход:** CRUD бордов; `addToBoard(boardId, placeId)`; статусы `BoardItem`.
**Собрать (можно резать на 4a/4b при упоре в контекст):**
- 4a: CRUD бордов, тематические борды, добавление/удаление места, статус item.
- 4b: визуальная раскладка (коллаж), умные метки на карточке (лучший сезон, вердикт, сложность), прогресс «посещено N из M».
**Definition of done:** место добавляется/удаляется, статус меняется, метки берутся из `Verdict`; тесты CRUD зелёные.
**Не загружать:** квиз, AI, монетизацию (только их типы).
