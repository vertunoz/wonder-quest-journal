# Part 7 — Триггерный движок (пуши)  (V2)

**Цель:** возвращать между поездками. Чистый сервис уведомлений.
**Зависит от:** `BoardItem`, `Place.decision.bestMonths`, граф друзей (Part 6).
**Контракт вход → выход:** правила → очередь уведомлений `{userId, type, payload}`.
**Собрать:** сезонное окно («пик через N недель для места из борда»), активность друзей, прогресс коллекции по региону, «возврат к мечте». Без манипулятивных дейли-стриков.
**Definition of done:** для тестового борда движок генерит корректный сезонный триггер по дате; юнит-тесты на каждое правило.
**Не загружать:** UI борда — только типы.
