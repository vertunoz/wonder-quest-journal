# Part 8 — Nearby live + оффлайн  (V2)

**Цель:** режим поездки — красивые места рядом + оффлайн-доступ.
**Зависит от:** `Place.coordinates`, геолокация.
**Контракт вход → выход:** `(lat,lng,radius)` → отсортированные `Place[]`; экспорт борда/маршрута в оффлайн.
**Собрать:** гео-запрос 30/100/300 км, сортировка по вердикту/расстоянию, оффлайн-кэш выбранного.
**Definition of done:** запрос по координатам возвращает места в радиусе (тест на радиус); оффлайн-набор доступен без сети.
**Не загружать:** прочие части.
