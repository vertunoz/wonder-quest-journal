# Part 9 — AI-ассистент по своей базе  (V2)

**Цель:** удобство (НЕ клин). Ассистент работает по нашей базе и профилю.
**Зависит от:** `Place`, `TasteProfile`. Использует существующую базу, ничего не выдумывает.
**Контракт вход → выход:** запрос пользователя → маршрут/сравнение/сезон из `Place[]`, отфильтрованных по `TasteProfile`.
**Собрать:** Trip Builder (даты/бюджет/дети→маршрут из базы), Compare, Best Season, Hidden Gems Nearby. Ответ всегда ссылается на реальные `Place.id`.
**Definition of done:** ассистент не возвращает места вне базы (тест-гард на «галлюцинации id»).
**Не загружать:** реализацию борда/карточки — только их типы.
