# Part 5 — Share / Clone борда  (V1)

**Цель:** виральная петля — друг получает ценность и клонирует борд к себе.
**Зависит от:** contracts: `Board`.
**Контракт вход → выход:** `Board` → (а) картинка для Stories, (б) публичная страница с `cloneBoard()`.
**Собрать:**
- Генерация share-картинки из борда (server-side render или canvas).
- Публичный read-only роут борда; кнопка «клонировать себе» создаёт копию у нового юзера.
- Атрибуция автора в клоне (для creator-петли позже).
**Definition of done:** клон создаёт новый `Board` с теми же items у другого `ownerId`; тест на cloneBoard.
**Не загружать:** всё остальное.
